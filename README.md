# Click Clack Hotel Web

### Prerequisites

```
Node.js
Firebase
```

### Installing

```
npm install
npm install gulp -g
```

## Running in dev mode

It will run a local dev server (http://localhost:3030) and open the first page (http://localhost:3030/public/index.html).

```
npm start
```

## Deployment

It will compile and deploy the local project to Firebase Hosting.

```
firebase deploy
```
